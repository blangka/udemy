# udemy python 강의 공부

## 강의 자료
100개의 프로젝트로 Python 개발 완전 정복를 통해 다양한 챕터의 파이썬 프로젝트 샘플을 진행하며 파이썬을 활용하는 repository 입니다.

## 🚀 Quick start
```shell
$ python3 main.py
```

## 1.basic
- 간단한 입출력 및 기본 프로젝트 생성 연습
- 파이참에서 디버깅하는 방법 
    - BreakPoint 설정한다.
    - Run -> Debug 'main.py' 를 실행한다.
